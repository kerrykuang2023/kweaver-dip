# **一、功能概述**
**一句话定义：**配置应用账户是新建/编辑数字员工流程中「知识配置」环节的前置能力，用于为数字员工配置访问业务知识网络所需的认证凭证，并完成数据查询权限的授权。

**功能目的：**目前所有数字员工共用一个应用账户，存在数据不隔离的安全隐患。当前需求解决了不同数字员工访问业务知识网络时的身份认证与权限管理问题。用户在选择业务知识网络前，需先配置应用账户作为访问凭证，并为应用账户授权可访问的业务知识网络及权限类型（数据查询权限），从而保障数字员工对业务知识网络的安全、可控访问。

**目标用户：**数字员工创建者、配置者。

**功能位置：**位于「新建/编辑数字员工」页面 → 左侧导航「知识配置」Tab 内，是业务知识网络选择的前置步骤。

# **二、典型使用场景**
## **场景一：首次为数字员工配置知识网络**
**触发条件：**用户在新建数字员工的「知识配置」页面，点击【选择知识】按钮，此时该数字员工尚未配置应用账户。

**问题原因：**数字员工访问业务知识网络需要通过应用账户进行身份认证，未配置应用账户无法建立访问通道。

**用户目标：**用户选择或新建一个应用账户，随后选择业务知识网络，并为应用账户配置对应知识网络的数据查询权限，最终完成知识配置。

## **场景二：切换已配置的应用账户**
**触发条件：**用户已为数字员工配置了应用账户和知识网络，但因业务调整需要切换到另一个 应用账户。

**问题原因：**不同 应用账户 对应不同的访问权限范围，更换业务方向或知识库归属时需要切换 应用账户。

**用户目标：**通过点击应用账户旁的【切换】按钮，重新打开应用账户选择弹窗，选择新的应用账户完成切换。

## **场景三：删除已配置的应用账户**
**触发条件：**用户决定移除当前数字员工的应用账户配置。

**问题原因：**应用账户可能已失效、被误配或不再需要。

**用户目标：**点击【删除】按钮后，系统弹出二次确定弹窗，明确告知删除后果（数字员工将无法访问已关联的业务知识网络），确定后清除应用账户，已选的业务知识网络可能无法正常访问。

# **三、关键概念**
**应用账户**

**定义：**应用账户是用于访问业务知识网络的身份认证凭证。

**功能作用：**它决定数字员工“能否访问”以及“能访问哪些”业务知识网络。每个数字员工绑定一个 应用账户，应用账户 上配置的权限决定了访问范围。

**背景：**业务知识网络存储在平台中，数字员工需要通过应用账户认证后才能查询知识内容。



**业务知识网络（BKN）**

**定义：**业务知识网络是 KWeaver DIP 平台中组织和管理业务领域知识的核心单元。

**功能作用：**数字员工基于关联的业务知识网络回答问题和执行任务。



# **四、功能结构 / 信息架构**
本功能位于「新建/编辑数字员工」页面的「知识配置」Tab 下，包含以下模块：



+ 知识配置主页面：展示当前数字员工关联的应用账户、已选的业务知识网络列表，提供【选择知识】入口
+ 选择应用账户弹窗：单选应用账户列表，支持搜索与新建
+ 新建应用账户弹窗：输入应用账户名称创建新的应用账户
+ 选择业务知识网络弹窗：多选知识网络列表，支持搜索
+ 配置应用账户权限弹窗：为应用账户授权业务知识网络的数据查询权限
+ 删除应用账户确定弹窗：二次确定删除操作及后果提示



**模块间关系：**

+ 知识配置主页面 → 【选择知识】→ 判断是否有配置应用账户
+ 无应用账户 → 选择应用账户弹窗 → 选择/新建应用账户 → 自动打开知识网络选择弹窗
+ 有应用账户 → 直接打开知识网络选择弹窗
+ 业务知识网络选择确定 → 自动弹出应用账户权限配置弹窗 → 确定授权 → 回到知识配置主页面
+ 知识配置主页面应用账户区域 → 【切换】→ 重新打开应用账户选择弹窗
+ 知识配置主页面应用账户区域 → 【删除】→ 弹出删除确定弹窗

# **五、操作流程**
## **流程一：首次选择业务知识网络（无**应用账户**）**
**入口：**从左侧导航进入「知识配置」Tab → 点击【选择知识】按钮。

**操作步骤：**

1. 点击【选择知识】，系统检测到当前未配置应用账户，自动弹出「选择应用账户」弹窗。
2. 在弹窗中搜索或浏览可用应用账户列表，单选一个应用账户。若列表中没有合适的应用账户，点击【新建】。
3. （可选）在「新建应用账户」弹窗中输入账户名称，点击【确定】完成创建。
4. 选择应用账户后点击【确定】，系统自动关闭应用账户弹窗，并打开「选择业务知识网络」弹窗。
5. 在知识网络列表中勾选需要关联的业务知识网络（支持多选），点击【确定】。
6. 系统自动弹出「配置应用账户权限」弹窗，展示所选的且未配置权限的业务知识网络，默认全部勾选。将要配置的权限类型为数据查询，有效期为永久有效。
7. 确定权限配置后，业务知识网络和应用账户信息显示在知识配置主页面。

**结果：**

+ 页面顶部显示当前应用账户名称标签（如 应用账户: xxx）及【切换】【删除】按钮。
+ 已授权的知识网络展示在「知识配置」Tab，每项可单独删除。

## **流程二：已有应用账户时追加知识网络**
**入口：**在「知识配置」Tab 中点击【选择知识】按钮。

**操作步骤：**

1. 点击【选择知识】，系统检测到已配置应用账户，直接弹出「选择业务知识网络」弹窗。
2. 勾选新增的业务知识网络，点击【确定】。
3. 系统弹出「配置应用账户权限」弹窗，为新增的业务知识网络配置权限。
4. 确定后回到知识配置主页面，列表更新。

## **流程三：切换应用账户**
**入口：**在「知识配置」页面的应用账户标签右侧，点击【切换】按钮。

**操作步骤：**

1. 点击【切换】，弹出「选择应用账户」弹窗。
2. 选择新的应用账户，点击【确定】。
3. 系统自动弹出「配置应用账户权限」弹窗，展示所选的且未配置权限的业务知识网络，默认全部勾选。将要配置的权限类型为数据查询，有效期为永久有效。
4. 确定权限配置后，业务知识网络和切换后的应用账户信息显示在知识配置主页面。

## **流程四：删除应用账户**
**入口：**在「知识配置」页面的应用账户标签右侧，点击【删除】按钮。

**操作步骤：**

1. 点击【删除】，弹出二次确定弹窗。
2. 弹窗标题为“确定删除应用账户”，正文提示：“删除后，数字员工将无法访问已关联的业务知识网络，确定要删除吗？”
3. 点击【取消】关闭弹窗，不做任何操作。
4. 点击【确定删除】，清除当前 应用账户。

**异常处理：**

+ 删除为不可逆操作（当前为前端状态清除），删除后，数据员工将无法查询所选业务知识网络的数据。

# **六、验证规则**
| **字段名称** | **必填性** | **输入类型** | **输入限制** | **条件逻辑** |
| --- | --- | --- | --- | --- |
| 应用账户 | 条件必填 | 单选弹窗 | 从已有 应用账户 列表中选择或新建 | 点击【选择知识】时，若未配置 应用账户 则必须先选择 |

| 应用账户 权限授权 | 自动 | 勾选列表 | 默认全选；权限类型为数据查询，有效期永久 | 选择知识网络确定后自动弹出 |
