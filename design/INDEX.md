# 特性设计及交互设计

本目录存放特性设计及交互设计，目录结构如下：

```
.
├── feature/              # 特性设计目录
│   └── studio/           # 数字员工平台特性设计
├── ux/                   # 交互设计目录
│   └── studio/           # 数字员工平台交互设计
```

## 读取规则

1. 首先根据用户需求，从 feature/ 目录下读取文档元数据，找到对应的特性设计
2. 仔细阅读特性设计文档，充分理解业务需求
3. 从 ux/ 目录下读取读取文档元数据，找到对应的交互设计
4. 仔细阅读交互设计文档，严格按照 Figma 设计稿还原，并遵循交互设计规则实现交互体验
